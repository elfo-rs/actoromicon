## Contributing

Install [mdBook](https://phaiax.github.io/mdBook/README.html):

```sh
cargo install mdbook
```

To build the book's HTML:

```sh
mdbook build
```

And to edit the book in pseudo-interactive "watch" mode:

```sh
mdbook watch --open
```
